# Cartella immagini

Salva qui le foto con questi nomi ESATTI e appariranno da sole sul sito:

| Nome file | Dove appare |
|-----------|-------------|
| `coppia.jpg` | Foto grande nella sezione "La Storia" (Alessandro & Lorena) |
| `galleria-1.jpg` | Galleria — riquadro 1 |
| `galleria-2.jpg` | Galleria — riquadro 2 |
| `galleria-3.jpg` | Galleria — riquadro 3 |
| `galleria-4.jpg` | Galleria — riquadro 4 |

## Come fare
1. Salva le foto dal telefono/Facebook sul computer
2. Rinominale come sopra (es. `coppia.jpg`)
3. Trascinale in questa cartella `img/`
4. Ricarica il sito: le foto ci sono!

Se un file manca, al suo posto resta il segnaposto colorato (nessun errore).

**Nota:** vanno bene `.jpg`. Se hai `.png` o `.jpeg`, rinominale in `.jpg` (o dimmelo e cambio io il codice).
