# Cumbia Kiz 💃🕺

Sito ufficiale della **Cumbia Kiz** — un nuovo stile di ballo nato nelle Marche (2026) dalla fusione di **cumbia** e **kizomba**, creato da **Alessandro & Lorena**.

## 🎯 Obiettivo
Far conoscere Alessandro e Lorena in tutte le Marche come i **creatori e maestri** della Cumbia Kiz, trovare allievi e far crescere corsi, stage e workshop.

## 📁 Contenuti del progetto
- `index.html` — sito vetrina (landing page, un unico file, nessuna dipendenza)
- `Piano-Cumbia-Kiz.md` — piano marketing in 8 passi (da 0 a notorietà regionale)

## 🚀 Come vedere il sito in locale
Apri semplicemente `index.html` con un doppio clic, **oppure** avvia un server locale:

```bash
python -m http.server 5500
```
Poi apri http://localhost:5500 nel browser.
